Можно вывести плавающую кнопку поверх игры и перед выходом из матча нажать кнопку разрыва соединений, тогда новые соединения быстро установятся и игроку не придется наблюдать за тем как курят в сторонке разработчики PUBG (наблюдать за дымом в лобби) забив на эту проблему.

Можно попробовать позвонить оператору и попросить увеличить время ожидания пакетов по TCP с соединениям до 1 часа, а лучше до 2-х. Если оператор это поправит, то Вам не нужна будет эта программа.

  An app to bypass the issue of long exits (smoke logging) after a PUBG match. This app resets TCP connections in the PUBG lobby, and the problem is solved. You can display a floating button over the game and press the "Disconnect" button before exiting the match. This will quickly establish new connections, and the player won't have to watch the PUBG developers smoke on the sidelines (watching smoke logging in the lobby), ignoring this issue.
  The crux of the problem is that when entering a match, the connection to the lobby server is idle and doesn't send packets. Many operators configure their equipment to wait for new packets for a short period, and after 4-5 minutes, the firewall deletes TCP connection data. Since matches usually last longer than 4-5 minutes, the lobby can't load correctly on the fly. You can try calling your operator and asking them to increase the TCP connection timeout to 1 hour, or better yet, 2 hours. If the operator fixes this, you won't need this program.

  You need to run the program as an administrator. The program needs these rights to see which connections are being used by the PUBG process and then reset them.
  Place the reset button in a convenient location and exit without any hassle.
  If the button disappears when you open the game, change the display mode to borderless windowed in the game settings.
